### Описание:
**"Автотесты"** -  тестирование интернет-магазина, проверяющее правильность работы продуктовых сценариев.<br>
Проект создавался в рамках прохождения курса "Школа разработки интерфейсов" Летних школ Яндекса.<br>

### Технологии:
- HTML5 / CSS3
- TypeScript
- Node.js
- Webpack
- Hermione
